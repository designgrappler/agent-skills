# Skill: Memory Indexer
## Category: Foundation
## Type: Skill

## Description
The "Persistent Cache" of the AgentOS. Distills tribal knowledge & user preferences into `MEMORIES.md` to ensure cross-session continuity.

## Rules:
- **Distill**: Append "Memory Hooks" to `MEMORIES.md` after major decisions/corrections.
- **Boot**: Mandatory read of memory index during session initialization.
- **Optimize**: De-duplicate and prune periodically for high signal-to-noise.

### Stats:
- Overhead: ~100-300T | Latency: Zero | Benefit: Zero-repeat error rate | Risk: Stale logic constraints.

## Trigger:
Tell Architect: "Index recent decisions to memory."
